Trustmetrics: WordPress plugin for review aggregation & analytics. Boost trust score with Google, Facebook reviews & gain insights.

== Description == 
Trustmetrics helps you collect and display genuine customer reviews from multiple platforms on your website. We are trusted by thousands of clients. Join Trustmetrics to read, write, and collect reviews from various platforms.

## Third-Party Service Usage

This plugin relies on the Trustmetrics API service for certain functionalities. The Trustmetrics API service is used under the following circumstances:

- Retrieving dashboard summaries.
- Managing user authentication and sessions.
- Fetching widgets and badges for display.

== Installation == 
1. Search for the Trustmetrics plugin and install it.
2. After activation, find the Trustmetrics menu in the left sidebar.
3. Click on the Trustmetrics menu to access the login page. Here, you can claim your free account or log in to your existing Trustmetrics account.
